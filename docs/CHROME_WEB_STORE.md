## Detailed Description

### Break the language barrier on Anthropic's AI courses.

Anthropic Academy offers free, world-class courses on Claude, prompt engineering, and AI safety — but only in English. SkillBridge translates the entire learning experience into 30+ languages so anyone in the world can learn.

**How it works:**
• Open any page on anthropic.skilljar.com
• Choose your language from the popup or sidebar
• The page translates instantly — UI labels, course content, lesson descriptions
• Embedded YouTube videos auto-enable translated subtitles
• Ask the AI Tutor questions about what you're learning

**Three-Tier Translation Engine:**
1. Static dictionaries with 560+ hand-tuned translations per language (instant)
2. IndexedDB cache for previously verified translations (instant)
3. Google Translate for remaining text, with Gemini 2.0 Flash verification for complex sentences

**AI Tutor powered by Claude Sonnet 4:**
A built-in chatbot that knows which course and lesson you're viewing. Ask questions in your language and get context-aware streaming responses.

**Premium Languages (curated dictionaries):**
Korean, Japanese, Chinese (Simplified), Spanish, French, German

**Standard Languages (Google Translate + AI verification):**
Chinese (Traditional), Portuguese, Italian, Dutch, Russian, Polish, Ukrainian, Czech, Swedish, Danish, Finnish, Norwegian, Turkish, Arabic, Hindi, Thai, Vietnamese, Indonesian, Malay, Filipino, Bengali, Hebrew, Romanian, Hungarian, Greek

**Privacy & Respect:**
• No API keys or accounts required
• No data collection or tracking
• Text is translated on-the-fly in your browser
• No course content is stored or redistributed
• Open source (MIT license)

**Not affiliated with Anthropic.** This is an independent community project built to make AI education accessible worldwide.

GitHub: https://github.com/heznpc/skillbridge

## Privacy Policy (short version for listing)

SkillBridge for Anthropic Academy does not collect, store, or transmit any personal data. All translations happen locally in your browser. The extension communicates only with:
- Google Translate API (for text translation, via background service worker)
- Puter.js (for AI Tutor chat and Gemini verification — no API keys, no accounts)
- Google Fonts (for CJK font rendering)
- YouTube IFrame API (for subtitle auto-enable)

No analytics, no tracking, no cookies. Full source code is available on GitHub under the MIT license.
