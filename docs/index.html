Why SkillBridge?

🌐

Real-time Translation

Translate entire course pages into your language with one click. Supports 30+ languages including CJK, Arabic, and more.

🤖

AI Learning Tutor

Ask questions about course material in your language. Get explanations, summaries, and help understanding complex topics.

💚

100% Free

Powered by Gemini 2.0 Flash + Claude Sonnet 4 via Puter.js. No API keys, no sign-up, no subscription. Just install and start learning.

🔒

Copyright-Safe

Translates on-the-fly in your browser only. Never stores, caches, or redistributes original course content.

How It Works

1

Install

Download from GitHub and load as an unpacked Chrome extension.

2

Visit Skilljar

Go to anthropic.skilljar.com and open any course.

3

Select Language

Click the globe icon and pick your preferred language.

4

Learn!

Read translated content and chat with the AI tutor.
